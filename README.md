## Sobre a Pokedex

Primiero , realizar um ``` git clone https://github.com/Cr-collab/Pokedex.git ```

Segundo ```cd  Pokedex ```

Terceiro, rode o  servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) em  seu Navegador e veja o resultado.

# Stacks Utilizadas

- React
- TypeScript
- Axios
- Saas
- React Icons
